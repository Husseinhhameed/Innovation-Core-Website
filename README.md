# 🚀 Innovation-Core-Website

**Nawat Al-Ebdaa — Smart Infrastructure & Security Solutions Corporate Website**

---

## 📖 Project Description

The **Innovation-Core-Website** is the fully static frontend of the official **Nawat Al-Ebdaa** corporate site, built with modern web standards using **HTML5**, **CSS3**, and **vanilla JavaScript**. This website is designed to effectively communicate the company’s expertise in smart infrastructure, security systems, network solutions, CCTV installations, and related technical services.

Optimized for performance and SEO, this lightweight site offers an engaging user experience across all devices without relying on backend technologies. It serves as a professional digital presence highlighting company values, service offerings, project workflows, and real-world case studies — complemented by accessible contact options.

Explore the live website here:  
🔗 [https://nawatalebdaa.com](https://nawatalebdaa.com)

## 🛠 Installation

This project is a static website and does not require any build tools or package managers.

To run locally:

1. Clone the repository  
   ```bash
   git clone https://github.com/your-username/Innovation-Core-Website.git
   ```
2. Navigate to the project folder  
   ```bash
   cd Innovation-Core-Website
   ```
3. Open `index.html` or `consultation.html` in your preferred web browser

Alternatively, deploy directly to any static hosting platform such as GitHub Pages, Netlify, or Vercel by linking the repository.
